# Web Browser Based TFTP File Transfer System

## Project Overview
This project implements a simple TFTP-based file transfer system using UDP socket communication.

The purpose of this project is to understand:
- UDP communication
- File transfer protocols
- Client-server architecture
- Basic network programming

## Development Environment
- Python 3
- UDP Socket
- GitHub

## System Structure
- Server receives file requests from client
- Client requests file name
- Server sends file data using UDP packets
- Client stores received file

## Main Features
- File Request
- File Download
- UDP Communication
- Simple Error Handling
